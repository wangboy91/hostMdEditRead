# Markdown Editor & Reader - Web版

这是一个基于Web技术的Markdown编辑器，无需安装任何软件，直接在浏览器中使用。

## 功能特性

- ✨ **实时预览编辑** - 支持分割视图、纯编辑、纯预览三种模式
- 📁 **文件加载和管理** - 支持加载多个本地Markdown文件
- 🎨 **语法高亮** - 支持代码块语法高亮显示
- 🌙 **深色/浅色主题** - 可切换的主题模式
- 💾 **本地存储** - 自动保存编辑内容到浏览器本地存储
- 🔍 **文件搜索** - 快速搜索和过滤文件
- 📤 **导出功能** - 支持导出为Markdown和HTML格式
- ⌨️ **快捷键支持** - 丰富的键盘快捷键

## 使用方法

### 1. 直接使用
在浏览器中打开 `index.html` 文件即可开始使用。

### 2. 本地服务器（推荐）
为了获得更好的体验，建议使用本地服务器：

```bash
# 使用 Python
python -m http.server 8080

# 使用 Node.js
npx serve .

# 使用 PHP
php -S localhost:8080
```

然后在浏览器中访问 `http://localhost:8080`

## 主要功能

### 文件管理
- **加载文件**: 点击"加载文件"按钮选择本地Markdown文件
- **文件列表**: 左侧显示已加载的文件列表
- **文件搜索**: 在搜索框中输入关键词过滤文件
- **清空列表**: 点击垃圾桶图标清空文件列表

### 编辑功能
- **实时预览**: 左侧编辑，右侧实时预览
- **语法高亮**: 代码块自动语法高亮
- **Tab缩进**: 支持Tab键插入4个空格
- **自动保存**: 每30秒自动保存到本地存储

### 视图模式
- **分割视图**: 同时显示编辑器和预览（默认）
- **纯编辑模式**: 只显示编辑器
- **纯预览模式**: 只显示预览

### 导出功能
- **保存Markdown**: 下载当前编辑的Markdown文件
- **导出HTML**: 将Markdown转换为HTML并下载

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` | 新建文件 |
| `Ctrl+O` | 加载文件 |
| `Ctrl+S` | 保存文件 |
| `Ctrl+E` | 切换编辑/预览模式 |
| `Tab` | 插入4个空格缩进 |

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式和布局
- **JavaScript ES6+** - 应用逻辑
- **marked.js** - Markdown解析
- **highlight.js** - 代码语法高亮

## 优势

相比Electron版本，Web版本具有以下优势：

1. **无需安装** - 直接在浏览器中使用
2. **跨平台** - 支持所有现代浏览器
3. **轻量级** - 不占用系统资源
4. **即时使用** - 打开即用，无需等待
5. **无网络依赖** - 可离线使用（除CDN资源外）

## 注意事项

1. 文件数据存储在浏览器本地存储中，清除浏览器数据会丢失
2. 大文件可能影响性能，建议单个文件不超过10MB
3. 某些浏览器可能限制本地文件访问，建议使用本地服务器

## 开始使用

1. 下载或克隆项目到本地
2. 在浏览器中打开 `index.html`
3. 点击"加载文件"选择Markdown文件
4. 开始编辑和预览

享受Markdown编写的乐趣！
