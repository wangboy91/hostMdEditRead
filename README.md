# Markdown Editor & Reader

一个跨平台的Markdown文档编辑和阅读工具，支持Mac和Windows系统，可以扫描磁盘上的所有.md文件进行操作。

## 功能特性

- ✨ **实时预览编辑** - 支持分割视图、纯编辑、纯预览三种模式
- 📁 **文件扫描和管理** - 递归扫描指定目录下的所有Markdown文件
- 🎨 **语法高亮** - 支持代码块语法高亮显示
- 🌙 **深色/浅色主题** - 可切换的主题模式
- 💾 **自动保存** - 支持保存和另存为功能
- 🔍 **文件搜索** - 快速搜索和过滤文件
- ⌨️ **快捷键支持** - 丰富的键盘快捷键
- 📊 **状态信息** - 显示字数统计、光标位置、文件大小等信息

## 系统要求

- **macOS**: 10.14 或更高版本
- **Windows**: Windows 10 或更高版本
- **内存**: 至少 512MB RAM
- **存储**: 至少 100MB 可用空间

## 安装和运行

### 开发环境运行

1. 克隆或下载项目到本地
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动应用：
   ```bash
   npm start
   ```

### 构建可执行文件

构建所有平台：
```bash
npm run build
```

构建特定平台：
```bash
# macOS
npm run build:mac

# Windows
npm run build:win
```

构建完成后，可执行文件将在 `dist` 目录中。

## 使用说明

### 基本操作

1. **新建文件**: 点击工具栏的"新建"按钮或使用快捷键 `Ctrl+N`
2. **打开文件**: 点击"打开"按钮或使用快捷键 `Ctrl+O`
3. **保存文件**: 点击"保存"按钮或使用快捷键 `Ctrl+S`
4. **扫描文件**: 点击侧边栏的"扫描文件"按钮选择目录

### 文件管理

- **扫描目录**: 选择一个目录后，应用会递归扫描其中的所有 `.md` 和 `.markdown` 文件
- **文件搜索**: 在搜索框中输入关键词来过滤文件列表
- **快速打开**: 点击文件列表中的任意文件即可快速打开

### 编辑模式

- **分割视图**: 左侧编辑器，右侧实时预览（默认模式）
- **纯编辑模式**: 只显示编辑器，适合专注写作
- **纯预览模式**: 只显示预览，适合阅读文档

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` | 新建文件 |
| `Ctrl+O` | 打开文件 |
| `Ctrl+S` | 保存文件 |
| `Ctrl+Shift+S` | 另存为 |
| `Ctrl+E` | 切换编辑/预览模式 |
| `Ctrl+F` | 扫描Markdown文件 |
| `Tab` | 在编辑器中插入4个空格 |

## 技术架构

- **框架**: Electron - 跨平台桌面应用框架
- **Markdown解析**: marked.js - 高性能Markdown解析器
- **语法高亮**: highlight.js - 代码语法高亮
- **UI**: 原生HTML/CSS/JavaScript - 轻量级用户界面
